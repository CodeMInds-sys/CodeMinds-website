1️⃣ مفهوم البرمجة

ما هي البرمجة؟

  • البرمجة = كتابة تعليمات للكمبيوتر ينفذها بالترتيب
  • الكمبيوتر ينفذ ما تقوله بالضبط (لا يفهم، يطيع فقط)

Compiler vs Interpreter:

  • Compiler: يترجم الكود كله دفعة واحدة لملف تنفيذي (مثل: C, C++)
  • Interpreter: ينفذ الكود سطر بسطر مباشرة (مثل: Python)
🐍 مثال توضيحي 
print("Hello")
print("World")
النتيجة:
Hello
World
2️⃣ Console App vs UI App

Console Application:

  • تطبيق يعمل في نافذة نصية (Terminal/CMD)
  • المستخدم يكتب نص، البرنامج يرد بنص

UI Application:

  • تطبيق له واجهة رسومية (أزرار، نوافذ، صور)
  • نفس المنطق لكن بشكل أجمل
لماذا نبدأ بـ Console؟
أسهل للتعلم والتركيز على المنطق البرمجي، بعدها نقدر ننقل نفس المنطق لـ UI
3️⃣ نموذج بناء أي تطبيق
القاعدة الذهبية:
INPUT → PROCESS → OUTPUT
إدخال → معالجة → إخراج

مثال حياتي: آلة البيع

  • تدخل نقود (input)
  • تختار منتج (process)
  • تستلم المنتج (output)

2️⃣ قواعد تسمية المتغيرات

القواعد الأساسية:

  1. يبدأ بحرف أو underscore _ (لا يبدأ برقم)
  2. يحتوي على حروف وأرقام و _ فقط
  3. حساس لحالة الأحرف: Age ≠ age
  4. لا يمكن استخدام كلمات محجوزة (مثل: print, if, for)

2️⃣ التحويل بين الأنواع (Type Conversion)

الدوال الأساسية:

  • int() → تحويل لعدد صحيح
  • float() → تحويل لعدد عشري
  • str() → تحويل لنص
  • bool() → تحويل لمنطقي

تحويل String إلى Integer:

🐍 str_to_int.py 
age_str = input("عمرك؟ ")      # "15"
age = int(age_str)              # 15
next_year = age + 1             # 16
print("العام القادم عمرك:", next_year)

# طريقة مختصرة
age = int(input("عمرك؟ "))
print("بعد 5 سنوات:", age + 5)

تحويل String إلى Float:

🐍 str_to_float.py 
height = float(input("طولك بالمتر؟ "))  # 1.75
print("طولك:", height, "متر")

تحويل Number إلى String:

🐍 num_to_str.py 
age = 15
message = "عمري " + str(age) + " سنة"
print(message)  # عمري 15 سنة

أمثلة متنوعة:

🐍 conversion_examples.py 
# int to float
x = 5

# float to int (يحذف الكسر)

w = int(z)      # 9

# string to int

# أي شيء to string
print(str(100))   # "100"
print(str(3.14))  # "3.14"
print(str(True))  # "True"
